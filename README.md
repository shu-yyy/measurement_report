# measurement_report

使用 Python 撰寫的 **自動化數據分析與報告生成工具**。  
適合處理校驗與量測數據，並自動生成包含數據摘要與圖表的 PDF 報告  

---

## ✨ 功能特色
- 讀取量測數據（CSV 檔案）
- 自動計算數據摘要（平均值、最大值、最小值）
- 繪製折線圖、直方圖
- 自動生成 PDF 報告（含中文支援）
